Введення даних для розрахунку середнього заробітку докладно

При розрахунку середнього заробітку для відпусток, відряджень і т.п. в таблиці представлена розшифровка суми, що враховується в розрахунку середнього заробітку:

• для всіх видів премій – кількість місяців базового періоду;
• для премій, що враховуються повністю – дата початку базового періоду;
• для річних премій – рік, за який нарахована премія.

Якщо ведеться облік за статтями фінансування, то для всіх сум показується їх розподіл за джерелами фінансування.

Для сум, які при розрахунку середнього заповнені автоматично згідно з раніше зареєстрованими в програмі нарахувань, відомості доступні тільки для перегляду.
Якщо суми вводилися при розрахунку середнього заробітку вручну, то відомості по ним тут допускається уточнювати.
