Форма призначена для заповнення відповідних кодів і номерів в табличних частинах документів.

У формі вводиться код для ПН та номер ВМД. Далі, для зазначених у документі товарів, виконується пошук відповідних елементів підпорядкованого довідника "Коди для ПН та номери ВМД". При успішному пошуку знайдені значення підставляються в документ.

Обробка так само дозволяє виконувати пакетне створення елементів довідника "Коди для ПН та номери ВМД" для обраних товарів. Для цього додатково проставляється прапор "Створювати коди для ПН та номери ВМД за заданими параметрами (якщо не знайдені)". Прапор доступний якщо у користувача є право на редагування даного довідника.
