Форма предназначена для заполнения соответсующих кодов и номеров в табличных частях документов. 

В форме вводится код для НН и номер ГТД. Далее, для указанных в документе товаров, выполняется поиск соответсвующих элементов подчиненного справочника "Коды для НН и номера ГТД". При успешном поиске найденные значения подставляются в документ.

Обработка так же позволяет выполнять пакетное создание элементов справочника "Коды для НН и номера ГТД" для выбранных товаров. Для этого дополнительно проставляется флаг "Создавать коды для НН и номера ГТД по заданным параметрам (если не найдены)". Флаг доступен если у пользователя есть право на редактирование данного справочника.
