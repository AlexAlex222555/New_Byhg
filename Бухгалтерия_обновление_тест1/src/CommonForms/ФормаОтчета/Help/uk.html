Для перегляду звіту необхідно натиснути кнопку Сформувати, попередньо задавши параметри (якщо вони передбачені). У клієнт-серверному варіанті роботи програми передбачено формування звітів у фоновому режимі.

Режим формування

  • Установіть прапорець Формувати відразу меню Ще, для того щоб звіт формувався автоматично відразу після відкриття, без натискання кнопки Сформувати.

Варіанти звіту:

  • У програмі може бути передбачено кілька варіантів звітів, що виконують схожий аналіз. Для їх перегляду передбачена кнопка Варіанти звіту.
    • Виберіть зі списку, що випадає, потрібний варіант звіту. Поточний варіант звіту відзначений прапорцем.
    • для перегляду списку інших варіантів звітів призначена команда Інші звіти. У вікні розміщений список схожих за змістом звітів.
      • В списку виводяться найменування та описи звітів. Список звітів згрупований. Передбачені групи, в яких виводяться схожі за змістом інші звіти того ж розділу програми.
      • Для переходу до іншого варіанту звіту натисніть на заголовок лівою кнопкою миші.
      • За допомогою відповідного прапорця можна Закривати вікно після переходу до звіту.

Збереження варіанту звіту

  • Натисніть Варіанти звіту - Зберегти варіант звіту, вкажіть Найменування та інші параметри.
  • Також можна скористатися командою Зберегти варіант звіту меню Ще - Варіанти звіту.

Збереження варіанту звіту в файл

  • Натисніть Ще - Варіанти звіту - Зберегти в файл для збереження варіанту звіту разом з призначеними для користувача настройками в архівний файл на комп'ютері. Вкажіть шлях до архівного файлу на комп'ютері. Разом з варіантом звіту за замовчуванням зберігаються і користувацькі настройки. Можна скасувати збереження всіх або деяких настройок за допомогою прапорців.
  • Ця можливість призначена для обміну настройками варіантів звітів між інформаційними базами ідентичної структури.

Завантаження варіанту звіту з файлу

  • Натисніть Ще - Варіанти звіту - Оновити з файлу для завантаження варіанту звіту, збереженого разом з користувацькими настройками в архівний файл на комп'ютері. Вкажіть шлях до архіву з варіантом звіту.
  • Ця можливість призначена для обміну настройками варіантів звітів між інформаційними базами ідентичної структури.

Настройки

  • По кнопці Настройки можна перейти до настройки параметрів звіту. Частина настройок (відбори) іноді виведені в шапку звіту.
  • Натисніть Ще- Інше- Стандартні настройки, для того щоб повернутися до настройок по умовчанню.
  • Натисніть Ще- Інше- Зберегти настройки, для того щоб записати поточні настройки звіту.
  • Натисніть Ще- Інше- Вибрати настройки, для того щоб скористатися раніше збереженими настройками звіту. У списку можна подивитися також настройки звіту, передані іншими користувачами.
  • Натисніть Ще - Інше - Поділитися настройками, для того щоб передати поточні настройки звіту іншим користувачам. Настройки передаються і без попереднього збереження. У списку, що відкрився, виберіть користувачів або групи користувачів, натисніть Поділитися. Якщо все в порядку, через деякий час програма виводить повідомлення про успішну передачу настройок.

Зміна умов відборів

  • За допомогою відповідної команди меню Ще можна переглянути і в разі потреби Змінити умови відборів по кожному полю.

Друк звіту

  • Сформований звіт можна роздрукувати, натиснувши кнопку Друк.
  • Для того щоб перевірити звіт і змінити параметри сторінки, натисніть Попередній перегляд.
  • За допомогою команди Ще - Параметри сторінки можна настроїти друк звіту.

Надсилання звіту електронною поштою

За допомогою кнопки Надіслати звіт електронною поштою можна:

  • Відправити зараз, для цього:
    • за допомогою прапорців виберіть один або кілька форматів, в яких необхідно відправити звіт.
    • при необхідності увімкніть прапорець Додатково упакувати в архів (. zip).
    • Натисніть Продовжити. Звіт буде прикріплений до електронного листа в якості вкладення.
  • Створити розсилку звітів;
  • Включити в існуючу розсилку звітів. Для цього зі списку розсилок виберіть потрібну.

Вибір варіанту звіту

  • Для завантаження раніше збереженого варіанту звіту скористайтеся командою Ще-Інше-Вибрати варіант звіту.
    • у списку варіантів звіту, натисніть кнопку Вибрати для переходу до потрібного варіанту звіту.

Зміна варіанту звіту

  • Натисніть Ще- Інше- Змінити варіант звіту. Можна додати поля, змінити параметри та настройки звіту.
  • Внесені зміни можна зберегти в інший варіант звіту.

Скидання настройок

  • Натисніть Ще- Інше- Стандартні настройки, для того щоб скинути користувацькі настройки.

Інші доступні дії

  • Для пошуку в звіті скористайтеся командою Знайти.
  • За допомогою кнопок Згорнути всі групи або Розгорнути всі групи можна приховати або показати групування звіту.
  • Для збереження результату звіту в файл натисніть, вкажіть найменування файлу і шлях для збереження.
  • За допомогою команд контекстного меню по правій кнопці миші можна:
    • Відкрити значення виділеної комірки;
    • Відфільтрувати записи за значенням обраного поля;
    • Упорядкувати:
      • за значенням вибраного поля (за замовчуванням);
      • Додатково - можна вибрати Поле для впорядкування звіту, Тип упорядкування - За зростанням або За спаданням;
    • Згрупувати:
      • за значенням вибраного поля (за замовчуванням);
      • Додатково - можна вибрати Поле для груповання; Тип груповання - Без ієрархії, Ієрархія, Тільки ієрархія;
    • Оформити:
      • настройте Елемент умовного оформлення звіту для обраного поля;
      • Додатково - можна настроїти оформлення для будь-яких полів звіту.
