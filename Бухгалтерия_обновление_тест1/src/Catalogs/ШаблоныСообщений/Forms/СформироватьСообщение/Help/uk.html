Призначений для вибору шаблонів повідомлень sms або електронних листів.

У лівій частині списку перераховані найменування шаблонів, в правій частині показується зміст шаблону.

Використання шаблону

  • Для вибору існуючого шаблону виберіть потрібний шаблон у списку, натисніть Сформувати.
  • Є можливість за допомогою відповідної кнопки відразу Сформувати і відправити sms-повідомлення.

Редагування шаблону

  • Для редагування шаблону двічі клацніть по потрібному найменуванню правою кнопкою миші. Також можна скористатися командою Ще - Змінити.

Введення нового шаблону

  • У списку завжди є пункт <Без шаблону>.
  • Для введення нового шаблону двічі клацніть по цьому пункту мишкою або виділіть його і скористайтеся командою Ще - Змінити.
