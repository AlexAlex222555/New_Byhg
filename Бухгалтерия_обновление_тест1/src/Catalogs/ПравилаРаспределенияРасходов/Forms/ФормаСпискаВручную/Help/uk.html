Форма списку вручну

У формі виводяться показники для розподілу витрат та їх значення.
При зміні робочого періоду виводяться показники діючі на даний період.

Кнопка Показувати тільки незаповнені показники дозволяє вивести показники для яких не вказані значення:

  • для показників, які вводяться щомісяця - повинні бути задані значення в вибраному робочому періоді;
  • для показників, які вводяться при зміні - має бути задано хоча б одне значення показника.
