Призначений для зберігання відомостей про версії файлів.

Відкривається командою Версії файлу на панелі навігації картки файлу.

У списку за допомогою кнопок панелі навігації можна:

  • Переглянути - відкрити версію файлу відповідним додатком для перегляду (також це можна зробити подвійним клацанням миші);
  • Відкрити картку - переглянути дані про версію файлу;
  • Зробити активною - встановити обрану версію файлу поточної. Команда доступна тільки для незайнятих файлів;
  • Показати відмінності - порівняти дві будь-які версії файлу, попередньо виділивши їх. Порівняння версій підтримується для файлів формату HTML, TXT, RTF, DOC, DOCX, ODT.
  • Записати версію файлу на комп'ютер за допомогою команди Зберегти як меню Ще. Вкажіть шлях на комп'ютері для збереження версії файлу.

Див. також:

  • Загальні відомості про файли;
  • Робота з формами.
