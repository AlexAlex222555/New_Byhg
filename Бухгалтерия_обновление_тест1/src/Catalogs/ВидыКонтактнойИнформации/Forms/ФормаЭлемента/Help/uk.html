Опис властивостей поля контактної інформації.

Поставка програми здійснюється напередвизначеними полями контактної інформації для деяких документів і списків. У визначених полів контактної інформації можна тільки поміняти настройки, інші реквізити заповнені за замовчуванням і захищені від редагування.

Типи контактної інформації

Програма дозволяє зберігати наступні типи контактної інформації:

  • Адреса - поле з таким типом контактної інформації призначене для зберігання адреси у вигляді довільного рядка або у вигляді шаблону адреси. Напередвизначені фактична та юридична адреси використовуються в друкованих формах документів.
  • Телефон - поле з таким типом контактної інформації призначене для зберігання телефонних номерів у вигляді довільного рядка, або відповідно до шаблону введення номера телефону. Напередвизначений телефон використовується в друкованих формах документів;
  • Адреса електронної пошти - поле з таким типом контактної інформації призначене для зберігання адреси електронної пошти. Реквізит використовується при відправленні електронних листів з програми.
  • Веб-сторінка - поле з таким типом контактної інформації призначене для зберігання адрес веб-сторінок. Дозволяє прикріпити гіперпосилання на інтернет-сторінки, після цього посилання будуть зберігатися в базі з можливістю переходу по них.
  • Факс - поле з таким типом контактної інформації призначене для зберігання номерів факсів у вигляді довільного рядка, або відповідно до шаблону введення номера факсу. Реквізит використовується в друкованих формах документів;
  • Інше - поле з таким типом контактної інформації може використовуватися для зберігання довільної текстової контактної інформації, наприклад, можна помістити інформацію про те, як дістатися до організації. Програма дозволяє в це поле ввести текст довжиною до 1024 символів.
  • Skype - поле з таким типом контактної інформації використовується для зберігання логіна Skype.

Додавання поля контактної інформації до списку або документа

Заповніть необхідну інформацію:

  • Виберіть зі списку Тип контактної інформації.
  • Введіть Найменування, з яким поле контактної інформації з'явиться у списку або документі.
  • Виберіть Групу, якщо вона не введена по умовчанню. Група відповідає документу або списку програми, в який додається нове поле контактної інформації.

Настройки полів контактної інформації

  • Увімкніть прапорець Обов'язкове заповнення, для того щоб програма перевіряла заповнення поля контактної інформації при введенні документів або списків. Збереження даних без заповнення цього поля буде неможливо.
  • Настройка Редагувати лише в діалозі використовується для типів контактної інформації Адреса, Телефон, Факс. Така настройка дозволяє вибрати спосіб введення цих видів контактної інформації:
    • У діалозі - дозволяє обмежити введення нового і редагування існуючого адреси, телефону або факсу шаблоном введення. При обраному значенні передбачена можливість ввести адресу у вільній формі. Це може знадобитися у виняткових випадках, наприклад, якщо дані адреси в будь-яких документах наводяться з помилкою, але для друку необхідно їх проставити. При цьому така адреса буде зберігатися в програмі з позначкою, що вона введена у вільній формі, і використовуватися з обмеженнями. Після уточнення адресу можна буде виправити, заповнивши за шаблоном.
    • В поле введення - якщо для адрес і телефонів не потрібно введення за шаблоном (наприклад, для іноземних адрес), то можна відключити введення в діалозі;
    • У діалозі і в полі введення - дозволяє використовувати обидва способи введення. Якщо правильне введення адреси не викликає труднощів, можна буде ввести адресу прямо в поле введення не відкриваючи діалогу, в разі труднощів можна буде відкрити шаблон і ввести адресу за допомогою шаблону.
  • Настройка Забороняти введення некоректної адреси використовується для полів Телефон або Адреса електронної пошти. Така настройка дозволяє заборонити введення телефонів або адрес електронної пошти, які заповнені не за встановленими правилами.
  • Увімкніть прапорець Дозволити введення декількох значень, для того щоб можна було ввести кілька значень. Наприклад, в організації або фізичної особи може бути кілька адрес електронної пошти або номерів телефонів, за якими можна зв'язатися з ними.

Особливості настройки поля Адреса

  • Включати країну до представлення - увімкніть прапорець, для того щоб країна вказувалась при автоматичному складанні програмою повної адреси в полі Адреса.

  • Зберігати історію змін - увімкніть прапорець, для того щоб зберігати історію змін, використовується не для всіх об'єктів програми, залежить від настройок розробника.

Міжнародний формат адреси

  • Для створення міжнародної адреси (наприклад, для валютних платежів), увімкніть прапорець Міжнародний формат адреси.

Особливості настройки поля Телефон і факс

  • Для того щоб використовувати Телефон з додатковим номером, увімкніть прапорець.
  • Для деяких типів телефонів (наприклад, мобільний телефон) не передбачено додаткових номерів.

Особливості настройки поля Адреса електронної пошти

  • Якщо встановлено тип поля Адреса електронної пошти, то програма перевіряє правильність вводу адреси електронної пошти. Увімкніть прапорець Забороняти введення некоректних адрес, в цьому випадку ввести некоректну адресу електронної пошти буде неможливо.

Особливості настройки поля Інше

  • За допомогою перемикача можна вказати Вид поля, для того щоб більш точно розташувати поле у вікні об'єкта:
    • Багаторядкове широке - включає кілька рядків;
    • Однорядкове широке - довгий рядок;
    • Однорядкове вузьке - якщо вид контактної інформації Інше використовується для введення текстових даних невеликої довжини, то розмір поля введення можна зменшити.

Див. також:

  • Введення контактної інформації;
  • Робота з формами.
