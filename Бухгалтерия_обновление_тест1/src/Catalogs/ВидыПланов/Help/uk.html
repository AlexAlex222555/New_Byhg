Основне

Для виду плану вказуються наступні реквізити:

  • Етап - на одному етапі планується одне з трьох: споживання, забезпечення або залишки. Від цього залежить які плани можна задати на етапі. Наприклад, на забезпечуючому етапі можна задати вид плану для купівель. Але неможна вид плану для планування продажів.
  • Операція - вказується для планів збирання (розбирання).
  • Вихідний або похідний (забезпечуючий). Вихідний для ручного введення даних. Похідний розраховується при зміні інших планів, від яких залежить. Настройка доступна для планів купівель, планів збирання( розбирання), плану виробництва. Для похідного (забезпечуючого) плану можна включити автоматичне формування і перерахунок планів при зміні потреб (настройка Заповнювати автоматично).
  • Використовувати для - визначає який план (продажів, купівель тощо) можна створювати за цим видом плану.
  • Заміщаючий або підсумовуючий - заміщуючі плани використовуються для змінного планування, коли нові плани витісняють/заміщають значення попередніх планів відповідного періоду. При "змінному" плануванні підтримується можливість редагування значень попередніх планів і підтримки планів в актуальному стані на певні періоди часу. При цьому попередні плани стають неактуальними. Підсумовуючий план підсумовує план з попереднім.
  • Із зазначенням плану оплат - дозволяє вводити плановані надходження або списання грошових коштів з планів продажів і купівель, розраховувати етапи оплат за вказаним в документі графіком. Якщо в сценарії зазначено що планування відбувається за кількістю і сумою.
  • Відображати за статтями бюджетування / відображати за показниками бюджету - якщо в сценарії планування визначена можливість відображення планів у бюджетуванні, то документи за Видом плану можуть формувати плановий оборот за статтями бюджетів для товарів і оплат. Для цього необхідно встановити відповідні прапори в групі Відображати за статтями бюджетування і вказати статті бюджетів, за яким відображається даний вид плану. Оплати можна відображати тільки якщо встановлений прапор Заповнювати план оплат. Статті бюджетів можуть бути уточнені безпосередньо в документі.

Для планів виробництва додатково задаються наступні параметри:

  • Тип процесу - можливі три варіанти: виготовлення, збирання; розбирання, утилізація; ремонт. Від того який процес обраний у вигляді плану, буде залежати які специфікації підбираються в документ плану. Наприклад, в план з типом "Ремонт", будуть підбиратися ремонтні специфікації.
  • Заповнювати діючі специфікації. На дату початку планування - діюча специфікація для кожного рядка визначається одна за датою початку планування, зазначеною в плані. На кожен період - діюча специфікація може змінюватися по періодах.
  • Планувати напівфабрикати - настройка задає спосіб планування забезпечуваних (не тих, що виробляються в процесі) напівфабрикатів. Автоматично - напівфабрикати плануються відкладено після проведення плану, одночасно з формуванням потреби в матеріалах. Вручну - напівфабрикати можуть бути заплановані додаванням в план окремими рядками або окремими планами.

Деталізація

Можна вказати ступінь деталізації плану (за підрозділом, призначенням, партнером, офертою, складом, форматом магазину, менеджером). Об'єднаний план по кожній деталізації дозволяє в одному документі вказувати безліч різних значень. Наприклад, скласти один документ – план з зазначенням всіх або кількох складів;

Настройки заповнення

Вихідні плани заповнюються за джерелами і формулами, Похідні (забезпечуючі) плани заповнюються за дефіцитом. Для Похідного (забезпечуючого) плану формули / джерела не налаштовуються.

Для Похідного (забезпечуючого) плану доступні довільні настройки відборів номенклатури потрапляють в план за дефіцитом.

Варіант заповнення (за формулою або джерелами) - простий варіант (за формулою) використовується для "тягнючою" схеми планування, коли цільовим є План продажів, а інші плани забезпечують його виконання, розширений варіант (за джерелами) використовується для довільного заповнення планів.

Правило заповнення за замовчуванням - можна задати і для варіанту заповнення за формулами і для варіанту заповнення за джерелами. Дозволяє вказувати схему співвідношення різних джерел при заповненні документів, налаштувати відбори, склад номенклатури, формули заповнення плану.

Обмеження на редагування правила заповнення - дозволяє заборонити редагування правила заповнення / формул в документі плану. Для розширеного варіанту заповнення, якщо заборона встановлена, кнопка настройки правила в документі плану буде прихована.
