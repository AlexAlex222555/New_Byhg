Правила отримання фактичних даних за показниками бюджетів

Правила призначені для настройки отримання фактичних даних за Показниками бюджетів при побудові бюджетних звітів і заповненні Примірника бюджету за фактичними даними.

Розділ джерела даних визначає, які джерела будуть використовуватися для отримання фактичних даних.

Можливі розділи отримання фактичних даних:

  • Оперативний облік – джерелами даних є Господарські операції з можливістю визначення Джерела суми для вибраної операції;
  • Регламентований облік і Міжнародний облік, джерелами даних є залишки по рахунках відповідного плану рахунків, при необхідності можна вибрати Кор.Рахунок обліку і Тип Підсумку (Оборот, Оборот по Дт, Оборот по Кт)
  • Довільні дані дозволяє настроїти довільну схему отримання фактичних даних.


Додатковий відбір дозволяє обмежити одержувані дані джерела.

Існує два способи визначення джерел даних для аналітик:

  • При включеному режимі Розширеної настройки заповнення аналітик вибирається спосіб і значення заповнення кожної аналітики при отриманні фактичних даних.
  • При відключеному режимі - заповнення аналітики визначається системою автоматично, на підставі типів даних наявних у джерелі.
