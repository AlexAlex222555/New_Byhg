Робочі ділянки

Довідник призначений для зберігання списку робочих ділянок, що використовуються на складських територіях. Робочі ділянки створюються для зручності роботи складських працівників при розміщенні нових товарів в складські комірках або для відбору товарів при відвантаженні. Робочі ділянки створюються для кожного складу або приміщення, в якому ведеться облік по складських комірках (довідково або зі зберіганням залишків в розрізі комірок). В якості робочих ділянок можуть бути, наприклад, визначено зони роботи автонавантажувача, зони відбору товарів вручну збирачем і т.д.

Список робочих ділянок може бути заданий при введенні інформації про нову складську територію або в момент генерації списку складських комірок. Кожна складська комірка може бути віднесена до певної робочої ділянки. В подальшому робочі ділянки використовуються для групування комірок при оформленні завдань на розміщення або відбір товарів при довідковому використанні комірок або для оформлення окремих завдань на розміщення і відбір товарів, якщо на складі (приміщенні) ведеться облік залишків товарів в розрізі комірок.

Формування робочих ділянок


Список робочих ділянок задається після запису інформації про склад.

  • У панелі навігації форми складу виберіть пункт Робочі ділянки.
  • З'явиться список визначених для складу робочих ділянок.
  • Для введення інформації про нову робочу ділянку натисніть на кнопку Створити.
  • Буде відкрита картка нової робочої ділянки.
  • Введіть найменування робочої ділянки.
  • Введіть інформацію про максимальну вантажопідйомності (вага, обсяг) робочої ділянки.
  • Натисніть на кнопку Провести та закрити.
  • Аналогічним чином введіть інформацію про всі інші робочі ділянки.


Якщо склад (складська територія) розділений по приміщеннях, то інформація про робочі ділянки вводиться для кожного приміщення. Введення інформації проводиться аналогічним чином.

Поділ складу (приміщення) на робочі ділянки не є обов'язковим. Це додаткова сервісна можливість, яка може бути використана на складі для розділення складу на робочі зони в разі використання різних варіантів відбору товарів на складі.
