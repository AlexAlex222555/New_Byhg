Містить загальну інформацію про валюти і призначений для додавання нових валют (рекомендується для коректного заповнення даних про валюту).

У Класифікаторі валют виводиться:

  • Код числовий - унікальний числовий код, прийнятий міжнародним стандартом, що служить для ідентифікації валюти при завантаженні курсів з мережі Інтернет.
  • Код символьний - прийнятий міжнародним стандартом, використовується як коротке найменування і основне представлення валюти.
  • Найменування валюти - повне найменування.
  • Завантажується з мережі Інтернет - якщо прапорець включений, то валюта може завантажуватися з мережі Інтернет.

Додавання валюти з класифікатора до списку валют

  • Виділивши одну або кілька валют, натисніть кнопку Вибрати.
  • Скористайтеся подвійним натисненням кнопки миші по валюті для додавання валюти в список валют.
