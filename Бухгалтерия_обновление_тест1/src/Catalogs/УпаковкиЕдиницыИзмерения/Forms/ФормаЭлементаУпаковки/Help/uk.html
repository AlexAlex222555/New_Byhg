Упаковка

Довідник призначений для реєстрації переліку одиниць вимірювань, визначених для позицій номенклатури, і упаковок товарів, які можуть використовуватися в якості альтернативних одиниць вимірювання номенклатури.

Передбачена можливість зазначення упаковок не тільки для конкретної позиції номенклатури, але і для декількох товарів. Для зазначення загального списку упаковок для декількох товарів використовується довідник Набори упаковок.

Приклад Підприємство торгує взуттям. Взуття поставляється в коробках, в яких знаходиться по 5 пар взуття. В довідник Набори упаковок вводиться набір упаковки Упаковки для взуття. Для набору упаковки вказується одиниця вимірювання - пара. У список упаковок для набору упаковки вводиться Коробка з кількістю 5 (пар). Для товарів, які відносяться до виду номенклатури Взуття, в полі Набір упаковок в картці позиції номенклатури вказується набір упаковки Упаковки для взуття.

Упаковка може бути трьох типів:

  • Кінцева - використовується, коли потрібно вказати кількість одиниць зберігання власника (набору або номенклатури) щодо одиниці вимірювання за класифікатором.
  • Складова - використовується, коли упаковка складається з інших упаковок, при цьому потрібно вказати кількість батьківських упаковок щодо одиниці вимірювання за класифікатором.
  • Разупаковка - використовується, коли треба вказати, що упаковка є частиною одиниці зберігання власника (набору або номенклатури), при цьому потрібно вказати кількість одиниць вимірювання за класифікатором щодо одиниці вимірювання власника (набору або номенклатури). Разупаковка можлива тільки для номенклатур / наборів, одиниці зберігання яких не є вимірними.
  • Товарне місце - використовується, якщо товар складається з декількох частин (товарних місць). Для кожної частини товару необхідно створити упаковку - товарне місце. Товарне місце не може бути складовою частиною інших упаковок будь-якого типу. Зазначення упаковок з типом "Товарне місце" можливо тільки в наступних документах: Прибутковий ордер на товари, Видатковий ордер на товари, Ордер на переміщення товарів (переміщення між складськими приміщеннями), Відбір (розміщення) товарів (адресний склад), Перерахунок товарів (по адресному складу), Ордер на відображення надлишків товарів, Ордер на відображення нестач товарів, Ордер на відображення псування товарів.

Приклад Є номенклатура "Шафа", упаковки використовуються для зберігання.
Одна шафа складається з наступних частин: корпус шафи - 1 шт, фасади шафи - 1 пара, фурнітура – 2 пачки. Для кожної частини потрібно створити окрему упаковку. Кожна з частин може зберігатися в окремій комірці.

Якщо в програмі ведеться облік залишків товарів у розрізі складських комірок, то зазначення упаковки для штучного товару є обов'язковим. У цьому випадку для кожної упаковки задаються розміри: вага, глибина, ширина, об"єм упаковки. Крім того, в довіднику задаються варіанти переупаковкок товарів (наприклад, паллета складається з 10 коробок, коробка складається з 10 штук). Ці дані необхідні для правильного розміщення товарів у складських комірках.

Заповнення інформації про розмір упаковки

Заповнення інформації про розмір упаковки

Список упаковок для товару може бути призначений індивідуально для конкретного товару. Якщо група товарів надходить в однакових упаковках, то список упаковок може бути об'єднаний в набір упаковок. Які упаковки застосовуються для конкретного товару (індивідуальні або набір упаковок) - визначається для кожного товару в картці позиції номенклатури.


У картці Упаковки введіть інформацію про розміри упаковки товару, вагу та об'єм упаковки. У разі використання підприємством багатооборотної тари, за допомогою прапора Використовується багатооборотна тара для неї необхідно уточнити відповідну характеристику.

При заповненні розміру упаковки можна використовувати раніше заповнений довідник Типорозміри упаковок. Після заповнення інформації відповідно до типових розмірів, можна відредагувати розміри упаковки товару.
