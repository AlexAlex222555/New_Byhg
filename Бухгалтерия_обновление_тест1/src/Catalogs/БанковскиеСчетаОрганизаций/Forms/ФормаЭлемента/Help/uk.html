Банківський рахунок організації

У довіднику зберігаються відомості про банківські рахунки організацій, що входять до складу підприємства. Для кожної організації може бути введено будь-яку кількість банківських рахунків.

Інформація про банківські рахунки заповнюється у формі конкретної організації. При введенні нового банківського рахунку організації вказується інформація, що характеризує банківський рахунок: місце відкриття, номер рахунку, валюта грошових коштів, банк, в якому відкрито рахунок і банк кореспондент для непрямих розрахунків, а також додаткова інформація, необхідна для коректного заповнення платіжних документів за обраним рахунком.

Якщо рахунок відкритий в Україні, для зазначення банку достатньо заповнити код рахунку у форматі IBAN . При введенні значення в це поле відбувається пошук банку в довіднику Банки і автопідстановка значень в інші поля. Якщо банківський рахунок не знайдений, то пропонується його вибрати з довідника Банки.
Якщо інформації про банк немає в класифікаторі, то її можна заповнити вручну. Поля для заповнення інформації про банк будуть доступні при встановленні прапорця Змінити реквізити банку.

Якщо рахунок відкритий за кордоном, то для зазначення банку необхідно заповнити поле SWIFT, національний код банку (при необхідності), назву та адресу банку.

Інформація про банківські рахунки контрагентів (клієнтів, постачальників та інших партнерів) зберігається в окремому довіднику Банківські рахунки.

Заповнення організації в картці банківського рахунку

  • Якщо новий банківський рахунок вводиться зі списку, в якому здійснено відбір по конкретній організації, то інформація про організацію заповнюється автоматично.
  • Якщо організація не заповнилася, то її треба обов'язково вказати. Кожен банківський рахунок обов'язково повинен відноситись до конкретної організації.
  • Вибір організації здійснюється в списку вибору довідника Організації. Для швидкого вибору можна вказати код ЄДРПОУ (ДРФО) або коротке найменування організації.
  • Вибір організації - власника банківського рахунку допускається тільки для нового елемента довідника Банківські рахунки організацій, після запису довідника, змінити організацію не можна.

Встановлення валюти


Вибір валюти здійснюється зі списку вибору довідника Валюти. Для швидкого вибору валюти можна вказати код валюти або її найменування.

Заповнення банку, в якому відкрито розрахунковий рахунок

Пошук інформації про банк здійснюється в довіднику Банки. МФО банку визначається автоматично за номером рахунку. Інформація в довіднику Банки заповнюється з Класифікатора банків.

При заповненні інформації про банк заповнюється інформація, яка точно відповідає запису про банк в класифікаторі банків. Якщо необхідно відкоригувати дані, то необхідно це зробити у формі відповідного банківського рахунку. Відкоригована інформація буде використовуватися при оформленні всіх платіжних документів по цьому банківському рахунку.

Для зміни даних про банк встановіть прапорець Змінити реквізити банку і введіть інформацію про нові реквізити банку. Дана інформація збережеться після запису інформації про банківський рахунок.

Заповнення банку при розрахунках за кореспондентськими рахунками, відкритими в інших банках

  • Встановіть прапорець Платежі проводяться через кореспондентський рахунок, відкритий в іншому банку.
  • У цьому випадку стає доступним для редагування реквізит МФО банку для непрямих розрахунків.
  • Введіть МФО.
  • За значенням поля МФО автоматично підбирається відповідний банк спочатку з існуючих елементів довідника Банки.

Заповнення інформації про платника


Інформація про платника заповнюється у формі банківського рахунку організації на сторінці Друк платіжних доручень. Інформація про платника автоматично заповнюється скороченим найменуванням організації після зазначення організації в банківському рахунку.

  • Для редагування інформації про платника встановіть прапор Редагувати текст найменування організації.
  • Змініть текст у полі Текст найменування організації у полі Платник.
  • Введений текст буде підставлятися в поле Платник при друці вихідних платіжних доручень.

Настройка виведення назви місяця прописом

  • У формі банківського рахунку організації на сторінці Настройка друку платіжних доручень виберіть в якості варіанту виведення місяця - прописом.
  • При друку вихідних платіжних доручень, в яких вказано цей банківський рахунок, інформація про місяць буде виводитися прописом.
  • Наприклад, дата платіжного доручення 27.09 буде виведена на друк, як 27 вересня.

Формування найменування банківського рахунку організації


Для банківського рахунку організації передбачено автоматичне формування найменування по умовчанню. При записі банківського рахунку автоматично заповнюється найменування, якщо воно не заповнене.

Найменування по умовчанню формується за формулою: Користувач може змінити сформоване найменування, змінивши його вручну або вибравши з випадаючого списку.

В списку вибору передбачено два варіанти найменування:

  • <Найменування банку>, <Найменування організації> (<Найменування валюти грошових коштів>).
  • <4 останні цифри рахунку> в <Найменування банку>, <Найменування організації>

Відключення контролю списання грошових коштів по конкретному банківському рахунку


Якщо списання грошових коштів за розрахунковим рахунком не контролюється, то встановіть прапорець Дозволити списання грошових коштів без "заявок на оплату". Після цього витрачання грошових коштів по банківському рахунку можна буде виконувати без обов'язкового введення документа Заявка на витрачання грошових коштів.
