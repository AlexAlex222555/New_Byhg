Довідник "Код номенклатури для ПН та номери ВМД"

Довідник призначений для зберігання інформації про коди для ПН та номери ВМД, за якими на підприємство коли-небудь надходили товари (послуги, роботи) кожного найменування. Код для ПН - це код УКТЗЕД для товарів, та код ДКПП для послуг. Номери ВМД вказуються довідково і можуть не заповнюватися.

При отриманні товарів (послуг, робіт), для яких ведеться облік по кодах для ПН та номерах ВМД, коди та номери вносяться до довідника, а потім зазначаються в документі постачання (Придбання товарів і послуг). Інформація про коди для ПН та номери ВМД, внесена до довідника, може бути використана при наступних поставках товару. При надходженні може бути також вказана інформація про код для пн, без заповнення номера ВМД.

У момент оформлення митної декларації коди УКТЗЕД і номери ВМД вказуються безпосередньо в документі і вносяться в довідник автоматично.

У момент поставки або в момент проходження митниці інформація про коди для ПН та номери ВМД реєструється в довіднику "Види запасів". У момент відвантаження товарів (оформлення документа "Реалізація товарів і послуг") інформація про списані коди для ПН і номери ВМД заповнюється автоматично: списується залишок по тому поєднанню коду для ПН і номера ВМД, яке має більш ранню дату поставки.

Інформація про коди УКТЗЕД або ДКПП та номери ВМД використовується при оформленні податкових документів (Податкових накладних, Додатків 2 до податкових накладних).
