Справочник "Код номенклатуры для НН и номера ГТД"

Справочник предназначен для хранения информации о кодах для НН и номерах ГТД, по которым на предприятие когда-либо поступали товары (услуги, работы) каждого наименования. Код для НН - это код УКТВЭД для товаров, и код ДКПП для услуг. Номера ГТД указываются справочно и могут не заполняться.

При получении товаров (услуг, работ), для которых ведется учет по кодам для НН и номерам ГТД, коды и номера вносятся в справочник, а затем указываются в документе поставки ("Приобретение товаров и услуг"). Информация о кодах для НН и номерах ГТД, внесенная в справочник, может быть использована при следующих поставках товара. При поступлении может быть также указана информация о коде для НН, без заполнения номера ГТД.

В момент оформления таможенной декларации коды УКТВЭД и номера ГТД указываются непосредственно в документе и вносятся в справочник автоматически.

В момент поставки или в момент прохождения таможни информация о кодах для НН и номерах ГТД регистрируется в справочнике "Виды запасов". В момент отгрузки товаров (оформления документа "Реализация товаров и услуг") информация о списанных кодах для НН и номерах ГТД заполняется автоматически: списывается остаток по тому сочетанию кода для НН и номера ГТД, которое имеет более раннюю дату поставки.

Информация о кодах УКТВЭД или ДКПП и номерах ГТД используется при оформлении налоговых документов (Налоговых накладных, Приложений 2 к налоговым накладным).
