Шаблони етикеток і цінників

Довідник призначений для реєстрації шаблонів цінників і етикеток, які використовуються в системі.
Для кожного виду номенклатури може бути застосований свій шаблон для друку цінника або етикетки. Передбачена також можливість завдання шаблону цінника (етикетки) для окремої номенклатурної позиції.

Для друку цінників і етикеток відповідно до розробленого шаблоном застосовується обробка Друк етикеток і цінників. Обробка може бути викликана з документів, що супроводжують поставку або зміна цін на товари.

Створення нового шаблону

Створення нового шаблону


Новий шаблон цінника або етикетки може бути створений заново або на підставі шаблону, передбаченого по умовчанню. При створенні нового шаблону етикетки (цінника) використовується помічник створення етикетки (цінника):

  • Натисніть на кнопку Створити.
  • Відкриється помічник створення етикетки (цінника).
  • Вкажіть найменування шаблону.
  • У полі Призначення вкажіть, який шаблон створюється: етикетка для товарів, цінник для товарів, етикетка серії номенклатури, етикетка для складських комірок, етикетка для доставки.
  • У полі Розмір стрічки виберіть розмір стрічки для друку цінника (етикетки).
  • У вікні перегляду показується прототип шаблону.
  • Для друку зразка етикетки (цінника) в полі Для чого виберіть позицію номенклатури (складську комірку, документ і т. д.) і натисніть на кнопку Надрукувати зразок.
  • Натисніть на кнопку Зберегти для запису шаблону.


Шаблон цінника можна відредагувати, додавши в нього будь-яке поле зі списку доступних полів. Для редагування шаблону в помічнику слід використовувати гіперпосилання Змінити в редакторі. При створенні або редагуванні шаблону використовуються ті стандартні правила роботи, які застосовуються при роботі з табличними документами.

Для кожного макета можна настроїти свої параметри друку: принтер, розмір паперу, поля відступу тощо.

Порада. Порада. Якщо при створенні нового шаблону відомо, що в нього слід додавати нові поля, яких немає у шаблонах, що поставляються, то при створенні шаблону слід використовувати команду Створити новий. У цьому випадку помічник використовуватися не буде, новий шаблон відразу відкриється в режимі редагування.
