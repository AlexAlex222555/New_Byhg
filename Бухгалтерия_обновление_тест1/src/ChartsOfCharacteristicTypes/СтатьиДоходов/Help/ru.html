Статьи доходов

Статьи доходов служат для классификации доходов предприятия по прочей деятельности (прочая неоперационная деятельность, финансовая деятельность и др.). Если предприятие формирует финансовый результат по направлениям деятельности (включен флаг Финансовый результат по направлениям деятельности в разделе НСИ и администрирование - Финансовый результат и контроллинг - Финансовый результат) - для статьи доходов можно указать способ распределения доходов, зарегистрированных по ней, между разными направлениями деятельности.
Аналитика доходов позволяет указать дополнительные признаки доходов, зарегистрированных по данной статье. В распределении доходов по направлениям деятельности аналитика доходов не учитывается.

Для оформления реализации ОС, объектов строительства или НМА, в качестве аналитики доходов необходимо выбрать соответствующий справочник: ОС / Объекты строительства или НМА
